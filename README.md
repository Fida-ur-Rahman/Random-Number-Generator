# 🎲 Random Code Generator

A simple and efficient web application that generates random codes instantly.  
This project is built using modern web technologies and provides a fast, user-friendly interface for generating unique random codes.

---

## 🚀 Features

- 🔢 Generate random codes instantly
- ⚡ Fast and lightweight
- 🎨 Clean and responsive UI
- 📱 Mobile-friendly design
- 🔁 Regenerate codes with one click

---

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Git & GitHub

---

## 📂 Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/yourusername/random-code-generator.git

# Navigate into the project folder
cd random-code-generator

# Install dependencies
npm install

# Start the development server
npm start
